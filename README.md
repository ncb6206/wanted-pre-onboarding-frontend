# wanted-pre-onboarding-frontend

원티드 프리온보딩 프론트엔드 - 선발 과제용 레파지토리입니다

## 지원자 성명

나규태

## 프로젝트의 실행 방법
```bash
npm install
npm start
```  

## 배포 사이트

[배포 링크](https://natest.shop)

## 디렉토리 구조

```
📦
├─ src
│  ├─ api
│  ├─ components
│  ├─ hooks
│  ├─ routes
│  │  ├─ root(Home)
│  │  ├─ signin
│  │  ├─ signup
│  │  └─ todo
│  ├─ styles
│  ├─ types
└─ └─ utils
```
